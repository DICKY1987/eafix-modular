  ---
  MT4 ↔ Python Communication Channels

  1. HTTP REST API (Primary Channel)

  MT4 side: services/desktop-ui/2299900003260118_ExpiryZones.mq4
  - Uses MT4's WebRequest() to call http://127.0.0.1:5001
  - Routes: /expiries.csv, /indicators.csv, /health, /spot

  Python side: services/desktop-ui/2099900129260118_expiry_service.py +
  2099900128260118_expiry_indicator_service.py
  - FastAPI server on port 5001
  - Serves expiry zones, signal states, and indicator summaries as
  JSON/CSV

  ---
  2. Windows DDE — Real-Time Price Feed

  MT4 side: MT4's built-in DDE server ("MT4" topic "QUOTE")

  Python side: services/data-ingestor/src/2099900111260118_ingestor.py
  - Polls MT4 DDE at 100ms intervals
  - Normalizes ticks and publishes to Redis channel eafix.data.prices
  - Enabled by default (enable_dde: True in 2099900109260118_config.py)

  ---
  3. File-Based Outbox (JSON Lines)

  Python side: services/desktop-ui/2099900132260118_outbound.py
  - Writes signals to {HUEY_OUTBOX}/{symbol}.jsonl

  MT4 side: EA expected to poll .jsonl files — no explicit reader found
  yet

  ---
  4. CSV File Polling

  Python side: Transport Router
  (services/transport-router/src/2099900202260118_router.py)
  - Watches directories: data/calendar-ingestor, data/execution-engine,
  etc.
  - MT4 or an EA writes CSV files; Python validates and routes them
  downstream via HTTP

  ---
  5. Redis Pub/Sub (Inter-Service Bus)

  Not a direct MT4 channel, but the Python backbone. Key topics:

  ┌───────────────────────────┬─────────────────────────────────┐
  │           Topic           │             Purpose             │
  ├───────────────────────────┼─────────────────────────────────┤
  │ eafix.data.prices         │ Price ticks (from DDE ingestor) │
  ├───────────────────────────┼─────────────────────────────────┤
  │ eafix.orders.validated    │ Order intents to execution      │
  ├───────────────────────────┼─────────────────────────────────┤
  │ eafix.execution.completed │ Fill reports back from MT4      │
  └───────────────────────────┴─────────────────────────────────┘

  ---
  6. TCP Socket (Disabled)

  Configured in 2099900109260118_config.py at localhost:5555 but
  disabled by default.

  ---

  Defined in mt4/helpers/2299900002260118_contract_parsers.mq4:
